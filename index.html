    Run commands in iTerm/Terminal from MacVim

    Project maintained by taiansu Hosted on GitHub Pages — Theme by mattgraham

    Seriously, what can this do?


    Installation

    • vim addon manager or Vundle (Best)
    • pathogen (Better)
    • manually (At least you use it)

    Usage

    • ,erRepeat the last execution or run current file ( act like ,ef for the first time )
    • ,efExecute the corresponding command of current file
    • ,elExecute the corresponding command of the line under cursor
    • ,ecClear iTerm's screen
    • ,use preloader commands ("spring" or "zeus")
